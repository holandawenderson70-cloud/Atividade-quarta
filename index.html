1. O que é Web Design e qual sua relação com Front-End?

Web Design é a área responsável pela criação do visual e da experiência do usuário em sites e aplicações web. Ele se relaciona com o Front-End, pois o desenvolvedor front-end transforma o design em código utilizando HTML, CSS e JavaScript.

2. Cite três princípios de design que melhoram a experiência do usuário.

Três princípios importantes são: clareza (facilidade de entender o site), consistência (manter padrões visuais) e hierarquia visual (organizar as informações por importância).

3. Qual a diferença entre Wireframe e Protótipo?

Wireframe é um esboço simples da estrutura do site, focado na organização dos elementos. Já o protótipo é mais detalhado, podendo conter cores, imagens e até simular interações.

4. Liste três ferramentas usadas para criar protótipos.

Três ferramentas populares são: Figma, Adobe XD e Canva.

5. Explique a função da tag <header> e dê um exemplo de uso.

A tag <header> é usada para representar o cabeçalho de uma página ou seção. Ela geralmente contém títulos, logotipos ou menus de navegação. Exemplo: o topo deste site com o título e o menu.

6. Qual a diferença entre uma tag em bloco e uma tag inline?

Tags em bloco ocupam toda a largura disponível e quebram a linha, como <div> e <p>. Tags inline ocupam apenas o espaço do conteúdo, como <span> e <a>.

7. Como aplicar CSS externo em um documento HTML?

O CSS externo é aplicado utilizando a tag <link> dentro do <head> do HTML, apontando para o arquivo .css.

8. Cite três propriedades CSS comuns e sua função.

color define a cor do texto, background-color define a cor de fundo e margin controla o espaçamento externo dos elementos.

9. O que é DOM e como o JavaScript interage com ele?

O DOM (Document Object Model) é a representação do HTML como objetos. O JavaScript interage com o DOM para alterar conteúdo, estilos e comportamentos da página.
